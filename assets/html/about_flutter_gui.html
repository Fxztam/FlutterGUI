About FlutterGUI

How did it start?

After attending the online Flutter Engage event 💙 I got so excited about the Flutter web stable release and I thought it's time to make something cool using it 👨‍💻. I've already tested the beta Flutter web release when it was announced but there wasn't official support and maintenance of flutter web plugins back then.

So I've started brainstorming and brainstorming and .... Nothing! 😬 I couldn't come up with any good idea. Throw the project to the bottom of the backlog.

A few weeks ago I was watching a video on the Flutter Dev youtube channel  📺 about Draggable Widget and I thought it would be cool to make Draggable apps inside the main app. So I've started working on that and after few hours I got the first dirty prototype working. Then I've started thinking: Why not turn it into an OS like User Interface. And here we are! 😎🤩

Why it looks familiar?

It wasn't my main focus to make an innovative design 🤷‍️ ... I was just trying to test how far I can push my technical knowledge and coding skills on getting something that works.

While implementing the UI I got inspiration from MacOS (The OS I am currently using), Ubuntu. and I also added some Windows XP flavor 🤪.

I'm not Something of a Designer Myself.

I got some help from my little brother 👶, who is a graphic designer, to design a new background and some unique icons.
